ZONE 5 — SCADA API (SCADA team · no changes from our side)
🔓 Decrypt — AES-256-GCM
SCADA team implements this (reference: DecryptExample.java)
1 Read byte[] from MQTT topic tms/scada/pas
2 Load key: ENV: SCADA_AES_KEY  (same key as Bridge server)
3 Extract IV = bytes [0 .. 11] (12 bytes)
4 Extract ciphertext = bytes [12 .. end]
5 AES-256-GCM decrypt → verify GCM tag (tamper check)
6 Parse JSON → update SCADA display
